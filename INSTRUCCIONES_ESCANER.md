# Instrucciones para Usar el Escáner de Código de Barras

## 📱 Escáner con Cámara del Celular

### ⚠️ IMPORTANTE - Permisos de Cámara

**El navegador SIEMPRE solicitará permiso para usar la cámara por razones de seguridad. Esto es normal y obligatorio.**

No es posible saltarse esta solicitud. Debes:
1. Permitir el acceso cuando el navegador lo solicite
2. Si denegaste el permiso por error, sigue la guía en `/app/GUIA_PERMISOS_CAMARA.md`

### Configuración Inicial

1. **Primera vez - Permitir Acceso a la Cámara:**
   - Al hacer clic en el botón de cámara (📷), el navegador mostrará un popup
   - Selecciona **"Permitir"** o **"Allow"**
   - La aplicación verificará el acceso automáticamente

2. **Si aparece un mensaje de error:**
   - Lee el mensaje completo - te dirá exactamente qué hacer
   - El mensaje incluye instrucciones específicas para tu navegador
   - Haz clic en **"Intentar de Nuevo"** después de permitir el acceso
   
3. **Si accidentalmente denegaste el permiso:**
   - Consulta la guía completa en: `/app/GUIA_PERMISOS_CAMARA.md`
   - Resumen rápido:
     - **Android Chrome:** Toca el candado → Permisos → Cámara → Permitir
     - **iOS Safari:** Ajustes → Safari → Cámara → Permitir

### Cómo Usar

1. **Abrir el Escáner:**
   - En la página de "Ventas"
   - Haz clic en el botón de cámara (📷) junto al campo de código de barras

2. **Escanear:**
   - Apunta la cámara al código de barras
   - Mantén el código dentro del marco verde
   - Asegúrate de tener buena iluminación
   - El escáner detectará automáticamente el código

3. **Resultado:**
   - El producto se agregará automáticamente al carrito
   - Verás una notificación de confirmación
   - El escáner se cerrará automáticamente

### Consejos para Mejores Resultados

- ✅ Mantén el teléfono estable
- ✅ Asegúrate de que el código esté bien iluminado
- ✅ Mantén una distancia de 10-20cm del código
- ✅ Evita reflejos o brillos en el código
- ✅ El código debe estar completo en el cuadro verde

### Problemas Comunes

**"Error al iniciar la cámara. Verifica los permisos"**
- Solución: Revisa los permisos de cámara en tu navegador

**"No se encontró ninguna cámara en el dispositivo"**
- Solución: Verifica que tu dispositivo tenga cámara y que no esté siendo usada por otra aplicación

**"Permiso de cámara denegado"**
- Solución: Ve a la configuración de tu navegador y permite el acceso a la cámara para este sitio

## 🔌 Escáner con Lector Físico USB

### Configuración

1. Conecta el lector de código de barras USB a tu computadora
2. El lector se comportará como un teclado
3. No requiere instalación de software adicional

### Cómo Usar

1. **Posiciona el Cursor:**
   - Haz clic en el campo "Escanea o ingresa código de barras"

2. **Escanear:**
   - Apunta el lector al código de barras
   - Presiona el gatillo del lector
   - El código aparecerá automáticamente en el campo

3. **Agregar al Carrito:**
   - Presiona Enter o haz clic en "Agregar"
   - El producto se agregará al carrito

---

## 💡 Recomendaciones

- **Para Ventas Rápidas:** Usa el lector físico USB (más rápido y preciso)
- **Para Movilidad:** Usa la cámara del celular (no requiere hardware adicional)
- **Iluminación:** Asegúrate de tener buena luz en ambos casos
- **Códigos Dañados:** Si un código no escanea, puedes ingresarlo manualmente

---

## 🆘 Soporte

Si tienes problemas con el escáner:
1. Recarga la página (F5 o Ctrl+R)
2. Verifica los permisos de cámara
3. Prueba con otro navegador (Chrome recomendado para móviles)
4. Verifica que el código de barras esté en buen estado
