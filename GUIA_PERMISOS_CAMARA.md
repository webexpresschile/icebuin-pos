# Guía para Habilitar Permisos de Cámara

## 📱 Android (Chrome / Firefox)

### Chrome Android

**Método 1: Durante la solicitud**
1. Cuando abras el escáner por primera vez, aparecerá un mensaje del navegador
2. Toca **"Permitir"** o **"Allow"**
3. ¡Listo! La cámara se activará automáticamente

**Método 2: Si ya denegaste el permiso**
1. Toca el **candado** 🔒 o el icono de información (ℹ️) en la barra de direcciones
2. Busca **"Permisos"** o **"Permissions"**
3. Encuentra **"Cámara"**
4. Cambia a **"Permitir"** o **"Allow"**
5. Recarga la página (desliza hacia abajo)
6. Vuelve a intentar abrir el escáner

**Método 3: Desde configuración del navegador**
1. Abre Chrome
2. Toca los **tres puntos** (⋮) en la esquina superior derecha
3. Ve a **Configuración** → **Configuración del sitio** → **Cámara**
4. Busca el sitio de FrozenPOS en la lista
5. Cambia el permiso a **"Permitir"**

### Firefox Android
1. Toca el **candado** en la barra de direcciones
2. Toca **"Editar configuración del sitio"**
3. Busca **"Cámara"**
4. Selecciona **"Permitir"**
5. Recarga la página

---

## 🍎 iOS (Safari)

### Safari iOS 15 y superior

**Durante la solicitud:**
1. Safari mostrará un popup cuando intentes usar la cámara
2. Toca **"Permitir"**
3. Si tocaste "No permitir" por error, sigue los pasos a continuación

**Desde Ajustes del iPhone:**
1. Abre la app **Ajustes** de tu iPhone
2. Desplázate y encuentra **Safari**
3. Toca en **Cámara**
4. Selecciona **"Permitir"** o **"Preguntar"**
5. Vuelve a Safari y recarga la página
6. Intenta abrir el escáner nuevamente

**Alternativa (por sitio específico):**
1. En Safari, ve a la página de FrozenPOS
2. Toca el botón **aA** en la barra de direcciones (arriba a la izquierda)
3. Toca **"Configuración para este sitio web"**
4. Busca **"Cámara"**
5. Selecciona **"Permitir"**

### Safari iOS 14 y anterior
1. Ve a **Ajustes** → **Safari** → **Configuración de Cámara**
2. Asegúrate que esté en **"Preguntar"** o **"Permitir"**
3. Recarga la página de FrozenPOS

## 💻 Escritorio (Chrome / Firefox / Edge)

### Google Chrome (Desktop)

**Durante la solicitud:**
1. Aparecerá un popup en la barra de direcciones
2. Haz clic en **"Permitir"**

**Si denegaste el permiso:**
1. Haz clic en el **candado** 🔒 en la barra de direcciones (junto a la URL)
2. Busca **"Cámara"**
3. Cambia a **"Permitir"**
4. Recarga la página (F5 o Ctrl+R)

**Desde configuración del navegador:**
1. Abre Chrome
2. Ve a **Configuración** (tres puntos → Configuración)
3. **Privacidad y seguridad** → **Configuración del sitio** → **Cámara**
4. Asegúrate que "Los sitios pueden solicitar usar la cámara" esté activado
5. Si FrozenPOS está en la lista de bloqueados, muévelo a permitidos

### Mozilla Firefox (Desktop)
1. Haz clic en el **candado** en la barra de direcciones
2. Haz clic en la **flecha** (>) junto a "Permisos"
3. Busca **"Usar la cámara"**
4. Desmarca **"Bloquear"** y marca **"Permitir"**
5. Recarga la página

### Microsoft Edge
1. Haz clic en el **candado** en la barra de direcciones
2. Haz clic en **"Permisos para este sitio"**
3. Busca **"Cámara"**
4. Cambia a **"Permitir"**
5. Recarga la página

## 🚫 Problemas Comunes y Soluciones

### "No se pudo acceder a la cámara"

**Posibles causas:**
1. **Otra aplicación está usando la cámara**
   - Cierra otras apps que puedan estar usando la cámara
   - En Android: Cierra otras pestañas del navegador
   - En iOS: Cierra la app de Cámara si está abierta

2. **La cámara está deshabilitada en el sistema**
   - Android: Ve a Ajustes → Apps → Navegador → Permisos → Cámara
   - iOS: Ve a Ajustes → Privacidad → Cámara → Safari

3. **Modo de navegación privada/incógnito**
   - Los permisos pueden no guardarse en modo incógnito
   - Sal del modo incógnito e intenta de nuevo

### "Tu navegador no soporta acceso a la cámara"

**Soluciones:**
- Actualiza tu navegador a la última versión
- Usa navegadores compatibles:
  - ✅ Chrome (Android/Desktop)
  - ✅ Safari (iOS/macOS)
  - ✅ Firefox (Android/Desktop)
  - ✅ Edge (Desktop)
  - ❌ No funciona en navegadores muy antiguos

### La cámara se enciende pero no escanea

**Soluciones:**
1. Asegúrate de tener buena iluminación
2. Mantén el código de barras dentro del marco verde
3. Limpia el lente de la cámara
4. Mantén el código de barras a 10-20cm de distancia
5. Intenta con otro código de barras para verificar

### Permiso concedido pero sigue sin funcionar

**Soluciones:**
1. **Limpia la caché del navegador:**
   - Chrome: Configuración → Privacidad → Borrar datos de navegación
   - Safari: Ajustes → Safari → Borrar historial y datos
2. **Cierra y vuelve a abrir el navegador**
3. **Reinicia tu dispositivo**
4. **Prueba en otro navegador** para descartar problemas del dispositivo

## ✅ Checklist Rápido

Antes de contactar soporte, verifica:

- [ ] Los permisos de cámara están en "Permitir" para el sitio
- [ ] No hay otras apps usando la cámara
- [ ] Tu navegador está actualizado
- [ ] Tienes conexión estable a internet
- [ ] El código de barras está limpio y legible
- [ ] Hay suficiente luz en el ambiente
- [ ] Has recargado la página después de cambiar permisos

---

## 📞 Soporte Adicional

Si después de seguir todos estos pasos sigues teniendo problemas:

1. Toma un screenshot del error
2. Anota qué navegador y versión estás usando
3. Anota qué modelo de dispositivo tienes
4. Describe exactamente qué pasa cuando intentas usar el escáner

**Alternativa temporal:** Si no puedes usar el escáner de cámara, siempre puedes:
- Usar un lector de código de barras USB (en computadora)
- Ingresar manualmente el código de barras en el campo de texto
